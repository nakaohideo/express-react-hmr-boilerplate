# Express-React-HMR-Boilerplate

This is a boilerplate for scaffolding MERN stack projects with livereload feature supported.

## Installation

```
npm install -g gulp
npm install -g jscs
npm install
```

## Build & Run

For development:
```
gulp
gulp serve
```

For production:
```
gulp build:production
npm start
```

## Test

```
npm test
```
